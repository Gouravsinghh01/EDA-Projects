# Netflix Exploratory Data Analysis (EDA)

This repository contains a data analysis project focused on the Netflix titles dataset. The goal of this project is to perform an exploratory data analysis (EDA) to uncover insights into the content available on Netflix, such as the distribution of content types, top genres, content ratings, and trends over time.

The analysis is presented through an interactive web application built with Streamlit, allowing users to explore the data dynamically.

## 🚀 Features

* **Interactive Dashboard**: A user-friendly web application for exploring the dataset.
* **Content Type Analysis**: Breakdown of movies vs. TV shows on the platform.
* **Temporal Analysis**: Visualizations showing the growth of content over the years.
* **Genre and Country Insights**: Charts to identify the most popular genres and the top content-producing countries.
* **Content Ratings**: Analysis of the distribution of content ratings (e.g., TV-MA, R, PG-13).
* **Interactive Visualizations**: Charts are created using Matplotlib, Seaborn, and Plotly for a rich and interactive user experience.

## 🛠️ Technologies Used

* **Streamlit**: For building the interactive web application.
* **Pandas**: For data manipulation and analysis.
* **Matplotlib**: For static data visualizations.
* **Seaborn**: For creating more aesthetically pleasing statistical graphics.
* **Plotly**: For interactive and dynamic plots.
